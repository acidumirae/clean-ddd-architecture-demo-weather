# AWS Kiro Auto-Configuration

This directory contains the configurations and steering intelligence for AWS Kiro.

## Directory Structure
*   `steering/` - Persistent markdown documents providing Kiro with project context, style guidelines, and rules.
*   `hooks/` - Event-driven hooks (JSON) attached to Kiro tools and IDE events for automation.
*   `specs_templates/` - Standardized templates for requirements and technical design.
*   `docs/` - Contains the `SDLC_PROCESS.md` explaining the workflow.

By modifying the items in this folder, you modify how Kiro interprets feature requests and governs code quality within this repository.
