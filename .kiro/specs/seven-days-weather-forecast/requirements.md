# Requirements Specification

## Goal
Get the 7 days weather forecast from the weatherapi.com API for the defined location and display it on a separate page.

## User Stories
*   As a user, I want to be able to provide city name input to get its 7 days weather forecast.
*   As a user, I want to be able to navigate to separate page to see the 7 days weather forecast.

## Acceptance Criteria
*   Condition A: There is a link to separate page to get the wether forecast
*   Condition B: There is an input field to accept user target location entry, eg. city and submit button to make the weather forecast request
*   Condition C: There is a result page or view to display the 7 days weather forecast for the entered city

## Non-Functional Requirements
*   Accessibility: Must comply with WCAG 2.1 AA.
*   Performance: API response time must be under 200ms.
